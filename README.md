# Аналитика пролонгации клиентских проектов (2023)

## Решаемые задачи
1. **Очистка и предобработка данных:** Устранение дубликатов, нормализация числовых значений, исключение остановленных проектов (статусы "стоп"/"end").
2. **Мэтчинг данных:** Объединение баз данных о проектах (`prolongations.csv`) и их финансовых показателях (`financial_data.csv`) по ID проекта.
3. **Расчет метрик:**
   * K1 (коэффициент пролонгации первого месяца) — отношение суммы фактически пролонгированных проектов в первый месяц к общей сумме проектов, подлежащих пролонгации.
   * K2 (коэффициент пролонгации второго месяца) — аналогичный расчет для клиентов, вернувшихся на второй месяц.
4. **Формирование отчетности:** Генерация структурированного Excel-файла с тремя листами («Весь отдел», «Менеджеры за год», «Менеджеры по месяцам») с применением корпоративного цветового оформления.

## Стек технологий
* **Python 3**
* **Pandas** — манипуляция данными, сводные таблицы, агрегация.
* **NumPy** — обработка пропущенных значений.
* **OpenPyXL** — форматирование и стилизация выходных отчетов (шрифты, заливка, границы).
